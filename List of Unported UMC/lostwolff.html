LOST UMC: WOLFF.TO

Although Wesnoth's server hosts add-on directories for Wesnoth releases going back to 1.0, not all of the earliest uploaded add-ons have been preserved. A number of the add-ons uploaded to the old wolff.to server (BfW 0.8 to 1.3) are missing.

In many cases, these missing add-ons exist in other versions. For example, there was a 1.2 version of The Dark Hordes; although it is not on the Wesnoth server today, there are ports to the other Wesnoth releases. (Actually, a 1.2 Dark Hordes is still available through UMC-Dev.)

However, some add-ons are not available from the Wesnoth server in any form. I've been going through the Internet Archive's Wayback Machine crawls of the old site, and I've turned up about three-dozen old add-ons that look as if they may be missing.
